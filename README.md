# capacitor-notes


```
npm install -g @ionic/cli

npm init @capacitor/app
```


### iOS

```
sudo gem install cocoapods

yarn add @capacitor/ios

npx cap add ios

yarn build

npx cap open ios

npx cap run ios
```

If `npx cap run ios` fails:

```
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
```
